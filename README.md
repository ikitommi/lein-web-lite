# Web lite Leiningen template

A simple, bare bones lein template for generating a webapp. Using Ring, Aleph
and Reitit. This template is meant for demonstration purposes.

## Setup

```sh
lein new web-lite hello-world
cd hello-world
lein run
```

## Creating a release

```sh
lein uberjar
java -jar
```
